Bienvenidos a EmpresaX!

Esta es la Web de EmpresaX, una empresa ficticia de desarrollo de software basado en software libre.

Notas de este diseño

Cumplimiento de estándares
Hemos seguido los siguientes estándares XHTML 1.0 Strict y CSS 2 para visualización con la mayoría de navegadores.
Diseño sin tablas
El diseño está realizado sin uso de tablas, permitiendo una mayor claridad y velocidad de representación.
Javascript+DOM
Hemos usado Javascript+DOM para la animación de tal forma que se representa correctamente en todos los navegadores.
Dos hojas de estilo
Se usa una hoja de estilo separada para el esquema de colores.
